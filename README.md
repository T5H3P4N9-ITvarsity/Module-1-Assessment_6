# Ecommerce Store

## Project Description
This project is an Ecommerce Store that showcases a catalogue of products. It includes a homepage with links to individual product pages, an About page detailing the business, and a Contact Us page for customer inquiries.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the homepage.
3. Navigate through the links to explore the About page, Contact Us page, and individual product pages.

## Features
- Homepage displaying a catalogue of 5 products.
- Individual product pages with detailed information.
- Navigation bar with links to Home, About, and Contact Us pages.
